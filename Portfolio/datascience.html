Summary

I have only recently switched over to the Data Science major and have a computer science background. I have taken two classes with data science programming and I have taken two classes for statistics. Two of the classes were in R, one in Python, and one with excel. One of my classes was Data Wrangling and Visualization: with this class I learned how to manipulate or sort and organize data and how to make appopriate graphs for the data given. For my statistics classes I learned about different statistical tests: t-tests, ANOVA, linear regression, chi-squared, along with non-parametric tests like Kruskal-Wallis. In my data science programming class I manipulated data with a Python library called Pandas and made graphs with Altair.
